Dit project is een webapplicatie waarin het spel Galgje (Hangman) wordt gespeeld met Engelse woorden. Het doel van het spel is om het verborgen woord te raden door letters te kiezen. Bij elke foute letter wordt een extra deel van het galgmannetje zichtbaar.

De applicatie is gebouwd met PHP en maakt gebruik van sessies om de spelstatus bij te houden.

De applicatie bevat de volgende functies:

De gebruiker kan kiezen tussen: Zelf een woord invoeren Een willekeurig woord laten genereren Het spel toont: Het aantal geraden letters Welke letters al gekozen zijn Visuele weergave van het galgmannetje: Bij elke fout wordt het mannetje verder “opgehangen” Duidelijk verschil tussen: Goede letters (groen) Foute letters (rood) Het spel eindigt wanneer: Het woord volledig geraden is (win) of je faalt
